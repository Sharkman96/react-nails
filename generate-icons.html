🎨 Генератор иконок SmartNails

Нажмите на кнопки для скачивания иконок нужного размера.

512x512


192x192


32x32 (favicon)


📋 Инструкция:

  1. Откройте этот файл в браузере (двойной клик)
  2. Скачайте все три иконки
  3. Скопируйте logo512.png и logo192.png в папку client/public/
  4. Для favicon используйте онлайн-конвертер (например, favicon.io) чтобы создать favicon.ico из favicon-32.png
  5. Пересоберите проект: cd client && npm run build
