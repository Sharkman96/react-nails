# Руководство по управлению контентом

## 📋 Добавление и редактирование услуг

### Расположение файла
Файл с услугами находится: `data/services.json`

### Структура услуги
```json
{
  "id": "уникальный-id",
  "key": "url-friendly-key",
  "name": {
    "ru": "Название на русском",
    "de": "Name auf Deutsch"
  },
  "price": {
    "ru": "35",
    "de": "35"
  },
  "description": {
    "ru": "Описание услуги на русском",
    "de": "Beschreibung auf Deutsch"
  },
  "icon": "scissors",
  "color": "#ff6b9d",
  "order": 1,
  "isActive": true
}
```

### Поля услуги

| Поле | Описание | Пример |
|------|----------|--------|
| `id` | Уникальный идентификатор | `"7"` или `"manicure-classic"` |
| `key` | URL-дружественный ключ (латиница, дефисы) | `"classic-manicure"` |
| `name.ru` | Название на русском | `"Классический маникюр"` |
| `name.de` | Название на немецком | `"Klassische Maniküre"` |
| `price.ru` | Цена (только число) | `"35"` |
| `price.de` | Цена (только число) | `"35"` |
| `description.ru` | Описание на русском | `"Профессиональный уход..."` |
| `description.de` | Описание на немецком | `"Professionelle Pflege..."` |
| `icon` | Иконка из Lucide | `"scissors"`, `"sparkles"`, `"palette"`, `"zap"` |
| `color` | Цвет акцента (HEX) | `"#ff6b9d"` |
| `order` | Порядок отображения | `1`, `2`, `3`... |
| `isActive` | Активна ли услуга | `true` или `false` |

### Доступные иконки
- `scissors` - ножницы (для маникюра)
- `sparkles` - блёстки (для дизайна)
- `palette` - палитра (для nail-art)
- `zap` - молния (для быстрых услуг)

### Пример добавления новой услуги

1. Откройте файл `data/services.json`
2. Добавьте новый объект в массив:

```json
{
  "id": "7",
  "key": "spa-manicure",
  "name": {
    "ru": "СПА-маникюр",
    "de": "SPA-Maniküre"
  },
  "price": {
    "ru": "55",
    "de": "55"
  },
  "description": {
    "ru": "Расслабляющий уход с массажем рук и парафинотерапией",
    "de": "Entspannende Pflege mit Handmassage und Paraffinbehandlung"
  },
  "icon": "sparkles",
  "color": "#8b5cf6",
  "order": 7,
  "isActive": true
}
```

3. Сохраните файл - изменения применятся автоматически

## 🖼️ Добавление изображений в галерею

### Расположение файлов
- **Данные галереи**: `data/gallery.json`
- **Изображения**: `public/images/gallery/`

### Требования к изображениям

| Параметр | Рекомендация |
|----------|--------------|
| Формат | JPG, PNG, WebP |
| Размер | 800x600 px (оптимально) |
| Соотношение сторон | 4:3 или 1:1 |
| Размер файла | До 500 KB |
| Именование | Латиница, дефисы, без пробелов |

### Шаг 1: Подготовка изображения

1. Оптимизируйте изображение (сжатие без потери качества)
2. Переименуйте файл: `french-manicure.jpg`, `gradient-design.png`
3. Скопируйте в папку `public/images/gallery/`

### Шаг 2: Добавление записи в галерею

Откройте `data/gallery.json` и добавьте новый объект:

```json
{
  "id": "7",
  "title": {
    "ru": "Нежный розовый",
    "de": "Zartes Rosa"
  },
  "category": "manicure",
  "imageUrl": "/images/gallery/gentle-pink.jpg",
  "color": "#ffc0cb",
  "order": 7,
  "isActive": true
}
```

### Структура записи галереи

| Поле | Описание | Пример |
|------|----------|--------|
| `id` | Уникальный идентификатор | `"7"` |
| `title.ru` | Название на русском | `"Нежный розовый"` |
| `title.de` | Название на немецком | `"Zartes Rosa"` |
| `category` | Категория работы | `"manicure"`, `"pedicure"`, `"design"`, `"extensions"` |
| `imageUrl` | Путь к изображению | `"/images/gallery/filename.jpg"` |
| `color` | Фоновый цвет (HEX) | `"#ffc0cb"` |
| `order` | Порядок отображения | `7` |
| `isActive` | Показывать ли | `true` или `false` |

### Категории работ
- `manicure` - Маникюр
- `pedicure` - Педикюр
- `design` - Дизайн ногтей
- `extensions` - Наращивание

## 🔄 Применение изменений

После редактирования JSON файлов:

1. **В режиме разработки** - изменения применяются автоматически
2. **На продакшене** - перезапустите сервер:
   ```bash
   # Остановите текущий процесс (Ctrl+C)
   # Запустите снова
   node server.js
   ```

## ⚠️ Важные замечания

### Формат JSON
- Используйте двойные кавычки `"` для строк
- Не забывайте запятые между элементами
- Последний элемент массива БЕЗ запятой
- Проверяйте JSON на валидность: [jsonlint.com](https://jsonlint.com)

### Переводы
- **Всегда** добавляйте оба языка (ru и de)
- Если перевод не готов, используйте английский как временный

### Изображения
- Используйте осмысленные имена файлов для SEO
- Оптимизируйте перед загрузкой (tinypng.com)
- Резервируйте оригиналы изображений

## 🆘 Решение проблем

### Услуги/галерея не отображаются
1. Проверьте `isActive: true`
2. Проверьте валидность JSON
3. Перезапустите сервер

### Изображение не загружается
1. Проверьте путь в `imageUrl`
2. Убедитесь, что файл в папке `public/images/gallery/`
3. Проверьте имя файла (регистр важен!)

### Ошибка JSON
Используйте онлайн-валидатор для проверки синтаксиса.
